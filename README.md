# QYNOTE

QYNOTE 是一个基于 HTML CSS JS Electron 开发的桌面笔记应用，提供简洁高效的笔记管理和待办事项功能。

## 功能特性

### 📝 笔记功能
- **Markdown 支持**：支持 Markdown 语法，提供编辑和预览模式切换
- **文件夹管理**：创建、重命名、管理笔记文件夹
- **笔记列表**：清晰的笔记列表，支持快速切换和搜索
- **笔记关联**：笔记可关联待办事项，方便任务管理

### ✅ 待办事项
- **添加/编辑/删除**：轻松管理待办事项
- **状态标记**：标记完成或未完成状态
- **统计功能**：显示剩余和总共的待办事项数量
- **批量操作**：清除已完成或清空所有待办事项

### 🎨 其他功能
- **主题切换**：支持亮色/暗色主题切换
- **响应式设计**：适配不同屏幕尺寸
- **本地存储**：数据保存在本地，确保隐私安全
- **简洁界面**：现代化的 UI 设计，专注于内容创作

## 技术栈

- **Electron**：跨平台桌面应用框架
- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript**：交互逻辑
- **Marked.js**：Markdown 解析

## 自打包安装（release也提供了打包好的版本）

### 环境要求
- Node.js 16.x 或更高版本
- npm 或 yarn 包管理器

### 安装步骤

1. 克隆或下载项目到本地
2. 进入项目目录
3. 安装依赖：

```bash
npm install
```

## 使用方法

### 启动应用

```bash
npm start
```

### 打包应用

```bash
npm run package
```

打包后的应用将生成在 `dist` 目录下。

## 应用界面

### 笔记视图
- **左侧**：笔记列表，显示所有笔记
- **右侧**：笔记编辑器，支持编辑和预览模式
  - **编辑模式**：Markdown 语法编辑
  - **预览模式**：实时预览渲染效果

### 待办事项视图
- **顶部**：添加新的待办事项输入框
- **中间**：待办事项列表，显示所有待办事项
- **底部**：统计信息和操作按钮

## 主要操作

### 笔记操作
1. **新建笔记**：点击左侧 "新建笔记" 按钮
2. **编辑笔记**：在右侧编辑器中输入内容
3. **保存笔记**：点击 "保存" 按钮
4. **删除笔记**：点击 "删除" 按钮
5. **切换视图**：点击 "编辑" 或 "预览" 切换模式

### 待办事项操作
1. **添加待办**：在输入框中输入内容并回车
2. **编辑待办**：双击待办事项进行编辑
3. **标记完成**：点击待办事项前的复选框
4. **删除待办**：点击待办事项右侧的删除按钮

### 文件夹操作
1. **打开文件夹管理**：点击左侧 "文件夹管理" 按钮
2. **创建文件夹**：在文件夹管理模态框中点击 "创建文件夹"
3. **重命名文件夹**：右键点击文件夹选择 "重命名"
4. **移动笔记**：拖拽笔记到指定文件夹

## 项目结构

```
QYNOTE/
├── index.html          # 主页面
├── main.js            # Electron 主进程
├── script.js          # 渲染进程逻辑
├── styles.css         # 样式文件
├── QYNOTE.ico         # 应用图标
├── package.json       # 项目配置
└── package-lock.json  # 依赖锁定文件
```

## 开发说明

### 开发流程
1. 修改代码
2. 运行 `npm start` 启动应用进行测试
3. 运行 `npm run package` 打包应用

### 主要文件说明

- **main.js**：Electron 主进程，负责窗口创建和应用生命周期管理
- **index.html**：应用界面结构
- **script.js**：应用逻辑，包括笔记管理、待办事项管理等
- **styles.css**：应用样式
